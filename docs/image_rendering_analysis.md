## 4. Z-Image Turbo 렌더링 능력 분석

### 4.1 잘 렌더링하는 항목 (8/10 이상)

| 항목 | 성공률 | 비고 |
|------|:------:|------|
| 의상 색상/패턴 | 10/10 | 폴카닷, 체크, 스트라이프 등 정확 |
| 의상 유형/스타일 | 9-10/10 | 오프숄더, 미니드레스, 스커트 등 정확 |
| 포즈/신체방향 | 8-9/10 | 앉기, 서기, 스케이팅 등 대부분 정확 |
| 배경/환경 | 8-9/10 | 벚꽃길, 엘리베이터, 욕실, 빙판 등 |
| 소재 질감 | 7-8/10 | 퍼 트림, 가죽 광택, 스타킹 반투명 |

### 4.2 약하게 렌더링하는 항목 (6/10 이하)

| 항목 | 성공률 | 원인 분석 |
|------|:------:|----------|
| 헤어 투톤/세부 | 5/10 | 투톤 헤어(오렌지 끝)를 단색으로 렌더링. 헤어 세부 변형은 모델 약점 |
| 텍스트/로고 | 5-6/10 | "Sprite", "英语" 등 일부 정확하지만 일관성 부족 |
| 소형 악세서리 | 6-7/10 | 뱅글, 팔밴드 등 프롬프트 후반부 항목 누락 빈번. 어텐션 디케이 영역 |
| 흰 장미 악세서리 | 0/10 | 두 method 모두 프롬프트에 명시했으나 Z-Image Turbo가 전혀 반영 안 함 |

### 4.3 프롬프트 길이와 렌더링 관계

spec-think-enzh 프롬프트가 std-nothink-enzh보다 평균 40-60% 더 김.
더 긴 프롬프트가 세부 항목(악세서리, 소재)에서 +6%p 우위를 보이지만,
일부 케이스(img4 드레스 동적 표현)에서는 짧은 프롬프트가 핵심 키워드에
어텐션을 집중시켜 더 나은 결과를 산출.

이는 Z-Image Turbo의 어텐션 메커니즘 특성:
- **0-75 토큰**: ~100% 어텐션 (핵심 요소 확실히 반영)
- **75-200 토큰**: 점진적 감쇠 (세부 요소 선택적 반영)
- **200+ 토큰**: ~30-50% 어텐션 (악세서리 등 누락 빈번)

## 5. 결론

1. **프롬프트 텍스트 이행률 차이가 이미지 렌더링에 반영됨** — spec-think-enzh의 텍스트 우위(~15%p)가 이미지에서도 확인(~6%p). 단, 이미지 차이 폭이 텍스트보다 작음.

2. **Z-Image Turbo의 렌더링 능력은 두 method 모두에서 높음** — 핵심 요소(의상, 포즈, 배경)는 80%+ 정확도로 재현. 차이는 세부 항목에서만 발생.

3. **Z-Image Turbo 자체 한계가 method 차이보다 지배적** — 헤어 투톤(5/10), 흰 장미(0/10) 등은 프롬프트 품질과 무관하게 미재현. 이는 프롬프트 개선으로 해결 불가.

4. **추천 유지** — 이미지 분석 결과가 기존 추천(spec-think-enzh 최고 품질, std-nothink-enzh 속도+품질)을 변경할 근거 없음.
